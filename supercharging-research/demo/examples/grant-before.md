# Grant Application: AI Agent Architectures (Before)

## The Old Way

This is what a grant writing session used to look like.

---

### Step 1: Open 14 browser tabs

- Google Scholar search for "multi-agent systems 2025"
- Half the results are paywalled
- Find a PDF on someone's personal site from 2019
- Get distracted by a tangentially related blog post
- Forget which tab had the good paper

### Step 2: Copy-paste into a Google Doc

> "Recent advances in multi-agent architectures have shown promise..."

Where did I read this? No idea. Add a [CITE NEEDED] and move on.

### Step 3: Meeting notes are somewhere

- Checked Slack -- nothing
- Checked email -- found a partial summary from Alice
- The actual recording is in a Zoom link that expired
- Reconstruct key decisions from memory

### Step 4: Write the literature review

"Several researchers have explored tool-use in language model agents [TODO]."

"The ReAct framework (Yao et al., [year?]) introduced..."

"As discussed in our team meeting on [date?], the approach of [person?] seems relevant."

### Step 5: Realize it's 2am

The related work section has 6 [TODO] markers, 3 [CITE?] notes, and references
a paper you're not sure actually exists. The methodology section is a bullet list
that starts with "basically we do the thing."
